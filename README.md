Le but du jeu est de survivre dans plusieurs salles remplies d'ennemis.

Le joueur doit se déplacer dans la salle, esquiver les ennemis et leur tirer dessus. Quand tous les ennemis d'une salle sont éliminés, les portes s'ouvrent et le joueur peut passer dans une autre salle.

Des objets peuvent apparaître au sol après avoir nettoyé une salle :
- le coeur permet de récupérer de la vie ;
- l'objet bleu augmente la vitesse du joueur.

La partie se termine si le joueur n'a plus de points de vie.

ATTENTION !!
Pour lancer le jeu dans visual studio code il faut :
Dans le terminal, écrire :
cd Codes
Puis pour lancer le menu écrire :
py main.py

Vous pouvez maintenant jouer.

DEPLACEMENTS
Si en QWERTY :
W : aller vers le haut
A : aller vers la gauche
S : aller vers le bas
D : aller vers la droite

Si en AZERTY :
Z : aller vers le haut
Q : aller vers la gauche
S : aller vers le bas
D : aller vers la droite

TIRS
Les touches de tir sont les memes en QWERTY et AZERTY :
I : tirer vers le haut
J : tirer vers la gauche
K : tirer vers le bas
L : tirer vers la droite

SAUVEGARDE:
En jeu, lorsqu'on appuie sur 'F5' une sauvegarde est automatiquement effectuée, pour la récupérer il suffit d'appuyer sur le bouton "CHARGER" du menu
